    Definir el estado de privacidad de un vídeo

    en su tema se describe cómo puede establecer el estado de privacidad de los vídeos que se distribuyen a YouTube mediante Brightcove Social.

    Al crear un destino de publicación de YouTube, una de las propiedades predeterminadas que se pueden establecer es la configuración de privacidad de YouTube. De forma predeterminada, hay tres opciones: Público, Privado y Sin listar. Cuando se publiquen vídeos en YouTube, la configuración de privacidad de todos los vídeos se establecerá en la configuración seleccionada.

    Es posible que tengas casos en los que la configuración de privacidad de YouTube para cada vídeo debería ser diferente. En este caso, se puede utilizar un campo personalizado asociado al vídeo para controlar la configuración de privacidad que permite que cada vídeo tenga una configuración de privacidad diferente. Cuando el vídeo se publica en YouTube mediante Brightcove Social, la configuración de privacidad se establecerá en el valor especificado por el valor del campo personalizado.

    Creación del campo personalizado

    Nota: Los campos personalizados sólo están disponibles para las cuentas Professional y Enterprise.

    Comience por crear en su cuenta el campo personalizado yt_privacy_status. Asígnele el tipo Lista. Para averiguar cómo añadir campos personalizados a una cuenta, consulte Crear campos de metadatos personalizados. Al modificar el valor de este campo personalizado podrá controlar el ajuste de privacidad de su vídeo en YouTube:

    yt_privacy_status Ajuste de privacidad del vídeo en YouTube
    private private
    public public
    unlisted unlisted
    value is null or unset public

    Consejo: Al asignar el tipo Lista al campo personalizado, los usuarios podrán modificar el estado a través de una lista desplegable, en lugar de tener que introducir manualmente el valor correspondiente.

    Asignación de un valor al campo personalizado

    Una vez creado el campo personalizado, su valor puede ajustarse desde el módulo Media.

    Edición del destino de publicación

    Edite el destino de publicación de YouTube y establezca la configuración de privacidad predeterminada en Usar campo personalizado YT_Privacy_Status.

    Publicación del vídeo

    Cuando se selecciona un vídeo y se selecciona el destino de YouTube, tendrá el estado de privacidad que se estableció mediante el campo personalizado.
